D2
Om D2: D2 är ett modernt diagramspråk optimerat för arkitekturdiagram. Kopiera koden nedan och klistra in i D2 Playground för att se och exportera diagrammet. 
direction: right

region: Region {
  style.fill: "#C7F9CC"
  basunderlag: Basunderlag
  urval: Urval A-B-C-D
  basunderlag -> urval
}

hubb: Hubb {
  style.fill: "#BFDBFE"
  standardpaket: Standardpaket P1-P7
  benchmark: Benchmark
  transport: Blind Relay
  policy: Policy-gate
}

spe: SPE {
  style.fill: "#DDD6FE"
}

externa: Externa {
  style.fill: "#FED7AA"
  sos: Socialstyrelsen
  ovriga: Ovriga
}

hubb.standardpaket -> region: 1. Standardpaket
region.urval -> hubb.benchmark: 2. Benchmark
hubb.benchmark -> region: Aterkoppling {style.stroke-dash: 3}
region.urval -> hubb.transport: 3. Krypterat
hubb.transport -> externa.sos: Blind relay
hubb.transport -> externa.ovriga: Blind relay
hubb.policy -> spe: 4. Federerad fraga
externa.ovriga -> hubb.policy: Begar analys {style.stroke-dash: 3}
spe -> region.basunderlag: Fraga
region.basunderlag -> spe: Resultat {style.stroke-dash: 3}

Så här gör du:

  1. Klicka på "Kopiera kod" ovan
  2. Klicka på "Öppna i D2 Playground"
  3. Markera all text i editorn (Ctrl+A) och ta bort den
  4. Klistra in koden (Ctrl+V)
  5. Klicka "Compile" för att se diagrammet
  6. Använd "Export" för att ladda ner PNG/SVG
